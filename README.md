# misc-tools

A collection of small command line tools

---

### clink
A Python utility for concatenating C source files.

**Purpose**
- Combine a main C file with reusable utility `.c` files (e.g. from a `C_Utils/` folder)
- Emit a single final `.c` file for compilation

More info in that folder's README

---

### pomodoro
A terminal-based Pomodoro timer

### mac_texter

Send Messages texts from the terminal
