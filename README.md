# misc-tools

A collection of small **command-line utilities**

---

### clink
A Python utility for concatenating C source files.

**Purpose**
- Combine a main C file with reusable utility `.c` files (e.g. from a `C_Utils/` folder)
- Emit a single final `.c` file for compilation

More info in that folder's README

---

### pomodoro
A terminal-based Pomodoro timer

---

### mac_texter
A macOS-only CLI tool for sending text messages.

**Purpose**
- Send Messages texts from the terminal
- Uses macOS GUI automation
- Maybe scripts that need to ping a phone

---

## notes
- Each tool is standalone
- Check each folder for a README with more domain specific instructions
